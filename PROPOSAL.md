## üéØ Problem Statement

### The Global Financial Inclusion Crisis

**1.4 billion adults worldwide** remain excluded from formal financial systems, particularly in developing economies. Traditional credit scoring systems create insurmountable barriers:

#### Traditional Banking Barriers
1. **No Credit History**: Underserved populations lack formal banking records
2. **Collateral Requirements**: Microenterprises cannot provide traditional collateral
3. **Geographic Exclusion**: Rural areas lack bank branches
4. **Complex Processes**: Paper-heavy applications discourage borrowers
5. **Bias & Discrimination**: Algorithmic and human bias in loan decisions
6. **Low Literacy Barriers**: Form-based applications exclude those with limited reading skills
7. **Harsh Rejections**: No explanation or improvement guidance when denied

#### The Sustainability Gap
- Traditional lenders ignore environmental and social impact
- Green businesses receive no preferential treatment
- No incentive structure for sustainable practices
- Lack of transparency in lending decisions
- No connection between institutional capital and microfinance

### Specific Focus: Indonesia & Southeast Asia

- **51% of Indonesian adults** are unbanked or underbanked (96M people)
- **65 million MSMEs** struggle to access capital despite high growth potential
- Rich alternative data sources (mobile payments, e-commerce) underutilized
- Strong government push for financial inclusion and green economy
- **$6+ trillion syndicated loan market** operates in silos from microfinance

## üìä Credit Scoring Methodology

### Three-Component Weighted System

Our algorithm evaluates applicants across **three dimensions**, weighted by real-world impact on underserved populations:

```
Final Score = (Traditional √ó 20%) + (Alternative √ó 50%) + (SDG √ó 30%)
```

**Approval Threshold**: 50/100  
**Green Bonus**: Up to -2% interest rate for high SDG scores

---

### Component 1: Traditional Credit Score (20% Weight)

**Rationale**: Limited weight for underserved populations without formal credit history.

#### Calculation:
```javascript
Base Score: 40 points (no credit history penalty)
+ Utility Payment History: Up to +20 points

Formula: 
  utilityScore = min(20, (onTimePayments / 24 months) √ó 20)
  
Max Traditional Score: 60/100
Weighted Contribution: 12 points (60 √ó 0.20)
```

**Why 20% Weight?**
- Traditional credit data often unavailable for target demographic
- Avoids penalizing unbanked individuals
- Still captures baseline financial responsibility (utility payments)

---

### Component 2: Alternative Data Score (50% Weight)

**Rationale**: Primary indicator for underserved populations. Rich data sources from digital economy.

#### Sub-Components:

**2A. Mobile Payment History (30 points max)**
```javascript
transactionScore = (transactionCount / 100) √ó 15
onTimeScore = (onTimePayments / totalTransactions) √ó 15

mobileScore = min(30, transactionScore + onTimeScore)
```

**Data Sources**: GoPay, OVO, Dana, ShopeePay transaction records

**2B. Business Presence (25 points max)**
```javascript
ratingScore = (googleMapsRating / 5) √ó 15
reviewScore = min(10, (reviewCount / 10) √ó 10)

businessScore = min(25, ratingScore + reviewScore)
```

**Data Sources**: Google Maps API, Gojek merchant ratings

**2C. Utility Payment Consistency (20 points max)**
```javascript
utilityScore = min(20, 
  ((electricityOnTime + waterOnTime + internetOnTime) / 72) √ó 20
)
```

**Data Sources**: PLN (electricity), PDAM (water), Telkom (internet) payment records

**2D. Social Proof (15 points max)**
```javascript
socialScore = min(15,
  (communityVouches √ó 3) + (businessReferences √ó 5)
)
```

**Data Sources**: Community leader endorsements, supplier references, customer testimonials

**Maximum Alternative Score: 90/100**  
**Weighted Contribution: 45 points (90 √ó 0.50)**

---

### Component 3: SDG Impact Score (30% Weight)

**Rationale**: Incentivize sustainable business practices. Align lending with UN SDG goals.

#### Environmental Factors (40 points max):

**3A. Carbon Footprint (15 points)**
```javascript
carbonScore = max(0, 15 - (carbonFootprint_tons / 2))
```
- Low emissions businesses score higher
- **SDG 13**: Climate Action

**3B. Renewable Energy (10 points)**
```javascript
renewableScore = renewableEnergy ? 10 : 0
```
- Solar panels, biogas, wind power adoption
- **SDG 7**: Affordable & Clean Energy

**3C. Waste Recycling (8 points)**
```javascript
recyclingScore = wasteRecycling ? 8 : 0
```
- Waste separation, recycling programs, composting
- **SDG 12**: Responsible Consumption

**3D. Organic Practices (7 points)**
```javascript
organicScore = organicPractices ? 7 : 0
```
- Organic farming, chemical-free production
- **SDG 2**: Zero Hunger (Sustainable Agriculture)

#### Social Impact Factors (30 points max):

**3E. Job Creation (10 points)**
```javascript
jobScore = min(10, jobsCreated √ó 2)
```
- **SDG 8**: Decent Work & Economic Growth

**3F. Women Employment (8 points)**
```javascript
womenScore = min(8, womenEmployed √ó 2)
```
- **SDG 5**: Gender Equality

**3G. Fair Wages (7 points)**

```javascript
fairWageScore = fairWages ? 7 : 0
```
- Above minimum wage, benefits, safe working conditions
- **SDG 10**: Reduced Inequalities

```javascript
communityScore = (impactScore_0to10 / 10) √ó 5
```

**Weighted Contribution: 21 points (70 √ó 0.30)**

---

### Final Score Calculation

```javascript
finalScore = round(
  (traditionalScore √ó 0.20) + 
  (alternativeScore √ó 0.50) + 
  (sdgScore √ó 0.30)
)

// Approval Threshold
approved = finalScore >= 50

// Loan Amount Eligibility
if (finalScore >= 80) loanAmount = 50000000  // Rp 50M
else if (finalScore >= 70) loanAmount = 30000000  // Rp 30M
else if (finalScore >= 60) loanAmount = 20000000  // Rp 20M
else if (finalScore >= 50) loanAmount = 10000000  // Rp 10M

// Interest Rate (Green Bonus)
baseRate = 18%  // Default
if (finalScore >= 80) baseRate = 12%
else if (finalScore >= 70) baseRate = 14%
else if (finalScore >= 60) baseRate = 16%

greenBonus = (sdgScore >= 50) ? 2% : (sdgScore >= 40) ? 1% : 0%
interestRate = max(10%, baseRate - greenBonus)
```

### Fairness & Bias Mitigation

1. **No Credit History Penalty**: Alternative data (50%) ensures unbanked aren't excluded
2. **Sustainability Rewards**: Green businesses get -2% interest and higher approval odds
3. **Transparent Criteria**: All factors and weights are public via Score Decomposition Dashboard
4. **Cultural Sensitivity**: AI trained on Indonesian context, understands informal economy
5. **Human Review**: Edge cases flagged for manual review before rejection
6. **Rejection with Dignity**: No harsh denials‚Äîeveryone gets actionable improvement plans

```solidity
    address borrower,

    uint256 alternativeScore,

    uint256 finalScore

// Disburse loan and record transaction
function disburseAndRecord(
    address borrower,
    uint256 amount,
    uint256 creditScore
) external payable onlyAuthorized

// Query borrower score history
function getBorrowerScore(address borrower) 
    external view returns (CreditScore memory)

```
